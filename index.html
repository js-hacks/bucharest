OPEN DATA CHALLENGES

This edition's theme is “Open Data” and we're getting together to solve challenges such as:

  • Unifying the available data from data.gov.ro
  • Using APIs for Transport/Infrastructure for Smart City solutions
  • Adopting/implementing APIs in the scope of local administration for public data
  • Access to anonymous medical data
  • Using APIs for notifications in emergency cases (earthquakes, floods, snow storms, etc.)

Have your own challenge you want to hack? Join us and let's build a solution together!
